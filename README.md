# ExperimentMind Pay

A trustworthy A/B testing analysis and decision platform for payment optimization experiments.

ExperimentMind Pay focuses on payment funnel experiments, including:

- payment attempt rate
- payment success rate
- revenue per user
- refund rate
- payment failure rate
- experiment quality diagnostics
- AI-assisted rollout recommendations

## Current Version

The current version is under development.

The first milestone is to build a reliable payment experiment data pipeline.
